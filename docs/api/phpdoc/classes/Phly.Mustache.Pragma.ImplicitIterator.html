IMPLICIT-ITERATOR pragma

When enabled, allows passing an indexed array or Traversable object to a section within a view, instead of an array of associative arrays or array of objects. By default, the tag {{.}} will render the current item in the array; you may specify an alternate tag name using the "iterator" option of the pragma:

{{%IMPLICIT-ITERATOR iterator=foo}}
{{#section}}
    {{foo}}
{{/section}}
